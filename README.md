# TestMVVMDaggerRxJava

## Simple app to try out common libraries

### Libraries used:

* RxJava
> To handle asynchronous tasks

* Dagger2
> To inject dependencies

* Retrofit
> To handle calls to the api

* Live Data
> To notify fragments of changings in the data

* Glide
> To load images 

* Espresso
> For UI testing

* Robolectric 
> For testing

* Mockito
> For testing
